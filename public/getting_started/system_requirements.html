System Requirements

Operating System  

Unix based Operating Systems (Linux and Mac OS) and MS Windows.

Web Server  

One of the following:

  • Nginx - Preferred
  • Apache

But, any webserver which supports PHP will work such as NginX, lighttpd, apache, cherokee.

Databases  

Depending on the project, one of the databases:

  • PostgreSQL >=9.4 (with citext extension) - Default and Preferred
  • MySQL >=5.7
PHP  

Spryker supports PHP >=5.6.

The preferred version is 7.1.

Support for PHP 5.6 will be stopped at some point and it is best to use a newer version for performance reasons.

SSL  

For production systems, a valid security certificate is required for HTTPS.
Redis  

version 2.8+

Elasticsearch  

version 2.4.x (not 5.x)
Jenkins (for cronjob management)  

version 1.7.x or 2.x
